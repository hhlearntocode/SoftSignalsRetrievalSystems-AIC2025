Enhanced TRAKE Sequence Search Algorithm:
How it works:
  • Enhanced Query Merging: Combines events with temporal markers
  • Pivot-Based Discovery: Finds best matching frames as sequence anchors
  • Frame-Number Windowed Search: Searches using frame numbers (keyframe_n) for efficiency
  • Matrix Similarity Computation: Batch computes all event-frame similarities simultaneously
  • Multi-Component Scoring: Advanced scoring with temporal, completeness, and consistency factors
Configuration Tips:
  • Search Window: Frame numbers to search around pivot (larger = more comprehensive)
  • Max Gap: Maximum frame number difference between consecutive events
  • Similarity Threshold: Minimum frame-to-event similarity (0.0-1.0)
  • Score Threshold: Minimum final sequence score to display (0.0-1.0)
